Case Study

StudyForge Study App

Interactive study app with API data and quiz generation.

Live Site GitHub

Objective

Build a study application that users can interact with by searching for topics, generating study quizzes, taking quizzes, and reviewing passed or failed answers.

Synopsis

StudyForge is similar to other online quiz providers and study websites that let users learn information through quiz generation. This app was made for students who want a more hands-on approach to learning information.

Process

Using the Wikipedia API and Open Trivia DB API, I created an app that allows users to search for quiz topics, enter quiz parameters, and save their progress through localStorage. I first focused on the main user flow, which was searching a topic, generating a quiz, answering questions, and reviewing results.

After that, I worked on displaying the quiz and results clearly so users could understand what they passed, what they missed, and what they needed to review. I also used localStorage so completed quiz results would stay available after the user finished a session.

Results

Users can learn information using this interactive study guide. StudyForge generates quizzes from searched topics so users can try again if they do not pass. After completing a quiz, users can review wrong answers and continue practicing until they pass.

The final project shows my ability to work with API data, JavaScript logic, localStorage, dynamic page updates, and user-focused interface design.
